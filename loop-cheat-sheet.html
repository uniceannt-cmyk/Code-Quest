∞Loop Cheat Sheet

What is Loop?

A loop let's you repeat a block of code multiple times—until a condition is met of for a set number of iterations

FOR LOOP

Use when you know how many times to repeat.

Js 
 for (let i = 0; i < 5; i++) {
  console.log(i);// 0 to 4
  }
  • i = 0: start
  • i < 5: condition
  • i++: increment
    • ✓Best for counting, iterating arrays, or fixed repetitions

    WHILE LOOP

    Use when you want to repeat as long as a condition is true

    Js 
        let i = 0;
    while (i < 5) {
      console.log(i);
      i++;
    }
    ✓Great for waiting on a condition or user input

    DO WHILE LOOP

    Runs at least once, then checks the condition

    Js 
        let i = 0;
      do {
         console.log(i);
           i++;
     } while (i < 5);
    ✓Useful when you want one guaranteed execution before checking
