Basic Operators:                                                    Comparison Operators:

+  : Addition                              ==   : Equal to
-  : Subtraction                           ===  : Equal value and type

/  : Division                              !==  : Not equal value or type 
(...) : Grouping                           >    : Greater than 
%  : Modulus (remainder)                   <    : Less than

-- : Decrement                             <=   : Less than or equal to

&   : AND                                                && : Logical AND

~   : NOT                                                !  : Logical Not

<<  : Left shift

Error Handling :

try...catch            : Used to handle errors in code.
throw                  : Throws a custom error.
Error object           : Represents an error in JavaScript.
try...catch...finally  : Handles errors and execute cleanup code.
try {

   //Code that may threw an error
   let result = 10 / 0; // This will throw a division by zero error
   console.log(result); // This line won't be executed

} catch (error){

    // Catch and handle the error
    console.log("An error occured:", error.message);

} finally {

    //Optiional cleanup code
    console.log("Cleanup code executed");
}
